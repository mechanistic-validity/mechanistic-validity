# Instrument Roadmap

Instruments to add beyond the current 58.

## Causal (A)
- EAP / Edge Attribution Patching (Syed et al.)
- ACDC automatic circuit discovery (Conmy et al.)
- Causal abstraction (Geiger et al.)
- Activation steering / representation engineering
- Circuit breaking (Zou et al.)

## Structural (B)
- SAE feature geometry / monosemanticity analysis
- Feature splitting / absorption detection
- Residual stream decomposition
- Attention pattern clustering

## Behavioral (D)
- Minimal pair evaluation
- Distribution shift robustness
- Task arithmetic (steering vector composition)
- Prompt perturbation sensitivity curves

## Representational (E)
- CKA (Centered Kernel Alignment)
- SVCCA
- Concept erasure (LEACE / RLACE)
- TCAV (concept activation vectors)
- Probing with control tasks (Hewitt & Liang selectivity)

## Measurement-theoretic (F)
- Effect size reporting (Cohen's d)
- Power analysis
- Multiple comparisons correction
- Ablation dose-response curves

## New family: Training Dynamics (G?)
- Phase transition detection
- Lottery ticket analysis
- Grokking stage classification
- Developmental trajectory tracking
- Circuit formation timelines
