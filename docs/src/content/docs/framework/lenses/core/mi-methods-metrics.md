# MI Methods (Redirected)

Methods are implementation techniques, not an evidence type. The metrics that were formerly on this page have been redistributed to their natural evidence-type categories:

- **Circuit discovery and attribution methods** (ACDC, EAP, SFC, RelP, etc.) are now in [MI Causal Metrics](/framework/lenses/core/mi-causal-metrics#6----circuit-discovery--attribution).
- **Steering and editing methods** (CAA, LEACE, RepE) are now in [MI Steering Metrics](/framework/lenses/core/mi-steering-metrics).

For technique-based lookup ("which metrics use ACDC?"), see the [Methods Index](/framework/evidence/methods-index).
