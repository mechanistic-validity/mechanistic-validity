# About

This section covers the motivation behind the framework, its intended scope and limits, contribution guidelines, and the changelog.

## Related repositories

Mechanistic Validity is split across three repositories:

- [**mechanistic-validity**](https://github.com/mechanistic-validity/mechanistic-validity) — the framework: metrics, calibrations, claim specs, and this docs site
- [**mechanistic-validity-lab**](https://github.com/mechanistic-validity/mechanistic-validity-lab) — experiment infrastructure: sweep runner, result tracking, Modal/RunPod integration
- [**mechanistic-validity-experiments**](https://github.com/mechanistic-validity/mechanistic-validity-experiments) — research: experiments using the framework on published circuits
